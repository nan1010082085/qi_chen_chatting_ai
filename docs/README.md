# AI 聊天助手 - 功能文档

欢迎使用 AI 聊天助手！这是一个基于 Vue 3 和 DeepSeek AI 的现代化聊天应用。

## 🚀 快速开始

1. **环境配置**：配置 DeepSeek API 密钥
2. **启动应用**：运行开发服务器
3. **开始对话**：创建新会话并开始与 AI 对话
4. **管理历史**：查看和管理历史对话记录

## 💡 主要特性

### 🤖 智能对话
- 基于 DeepSeek AI 模型的智能对话
- 支持流式响应，实时显示回复
- 支持思考过程展示（DeepSeek R1 系列）

### 💾 数据持久化
- 基于 IndexedDB 的本地存储
- 自动保存对话历史
- 支持多会话管理

### 🎨 现代界面
- 响应式设计，适配各种设备
- 基于 TDesign 的美观 UI
- 支持明暗主题切换

### 🛠️ 开发友好
- TypeScript 全面支持
- 完整的 ESLint 和 Prettier 配置
- 基于 Vite 的快速构建
